CSVulnWeb - ASP.NET Core 漏洞测试入口

命令执行漏洞 (CmdExec)

子类继承调用

子类继承 - 外部参数命令执行 GET /cmd/subclass?cmd=whoami 子类继承 - 硬编码命令执行 GET /cmd/subclass/hardcoded

接口实现类调用

接口实现 - 外部参数命令执行 GET /cmd/interface?cmd=whoami 接口实现 - 硬编码命令执行 GET /cmd/interface/hardcoded

代码执行漏洞 (CodeExec)

子类继承调用

子类继承 - 外部参数代码执行 GET /code/subclass?code=1+1 子类继承 - 硬编码代码执行 GET /code/subclass/hardcoded

接口实现类调用

接口实现 - 外部参数代码执行 GET /code/interface?code=1+1 接口实现 - 硬编码代码执行 GET /code/interface/hardcoded

SSRF漏洞

子类继承调用

子类继承 - 外部参数SSRF GET /ssrf/subclass?url=http://example.com 子类继承 - 硬编码SSRF GET /ssrf/subclass/hardcoded

接口实现类调用

接口实现 - 外部参数SSRF GET /ssrf/interface?url=http://example.com 接口实现 - 硬编码SSRF GET /ssrf/interface/hardcoded

跨文件函数调用 (FuncCall)

跨文件 - 命令执行 GET /funccall/cmd?cmd=whoami 跨文件 - 代码执行 GET /funccall/code?code=1+1 跨文件 - SSRF GET /funccall/ssrf?url=http://example.com 跨文件 - 通过子类调用 GET /funccall/child?cmd=whoami 跨文件 - 硬编码测试 GET /funccall/hardcoded

参数说明：cmd = 命令参数, code = 代码参数, url = 目标URL

所有漏洞均通过 父类VulnParent → 子类/接口实现类 → 路由层 构造完整调用链
