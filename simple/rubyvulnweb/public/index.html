Ruby on Rails 漏洞测试入口

1. 命令执行 (Command Execution)

使用 backticks / system() 执行系统命令

直接调用(外部参数) 直接调用(whoami) 子类继承调用 模块混入调用 硬编码场景 危险命令(外部参数)

2. 代码执行 (Code Execution)

使用 eval() 执行任意 Ruby 代码

直接调用(外部参数) Dir.glob调用 子类继承调用 模块混入调用 硬编码场景 system调用

3. SSRF (Server-Side Request Forgery)

使用 Net::HTTP 发起服务端请求

直接调用(外部参数) AWS元数据(外部参数) 子类继承调用 模块混入调用 硬编码场景

4. 跨文件函数调用 (Cross-File Function Call)

跨文件函数调用链中的漏洞触发

5. 漏洞调用链路径汇总

所有漏洞路径的深层调用链

类型场景调用链
命令执行子类继承参数 → CmdExecSubClass → VulnChild → VulnParent.vuln_cmd_exec → backticks
命令执行模块混入参数 → CmdExecModuleImpl.include(VulnModule) → module_cmd_exec → backticks
代码执行子类继承参数 → CodeExecSubClass → VulnChild → VulnParent.vuln_code_exec → eval
代码执行模块混入参数 → CodeExecModuleImpl.include(VulnModule) → module_code_exec → eval
SSRF子类继承参数 → SsrfSubClass → VulnChild → VulnParent.vuln_ssrf → Net::HTTP
SSRF模块混入参数 → SsrfModuleImpl.include(VulnModule) → module_ssrf → Net::HTTP
