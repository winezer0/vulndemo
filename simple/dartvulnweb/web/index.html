调用链说明

图例：

外部参数 = 通过URL query parameter传入的可控数据
硬编码 = 代码中固定的漏洞触发数据
子类继承 = 通过 VulnChild extends VulnParent 继承链触发
接口实现 = 通过 VulnInterface + VulnParent 组合实现触发
跨文件 = 通过 func_call.dart 中的独立函数触发

完整调用链示例：

外部请求参数 → server.dart路由 → 子类/接口实现类 → 跨文件函数 → 父类漏洞方法 → Process.run / eval / http请求
